# Homebrew Tap

Homebrew formulas for [@lox](https://github.com/lox) tools.

## Usage

```bash
brew tap lox/tap
brew install <formula>
```

Or install directly:

```bash
brew install lox/tap/<formula>
```

## Available Formulas

- **slack** - CLI for Slack ([source](https://github.com/lox/slack-cli))
